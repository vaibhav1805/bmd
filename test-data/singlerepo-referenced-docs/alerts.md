# Notification System (Alerts)

## Overview

Manages all notifications across the platform.

## Integration Points

- Triggered by [User Authentication](auth.md) for login alerts
- Triggered by [Payment Processing](payments.md) for transaction confirmations
- Events tracked in [Analytics](metrics.md)
- System design documented in [architecture.md](architecture.md)

## Channels

Multiple delivery channels available:
- Email
- SMS
- Push notifications
- In-app messages

See [overview.md](overview.md) for complete system description.
