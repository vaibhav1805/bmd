# System Overview

Welcome to our platform documentation. This system is built from several key components:

- [User Authentication](auth.md) - Handles user login and account management
- [Payment Processing](payments.md) - Manages all financial transactions
- [Notifications](alerts.md) - Sends alerts and messages to users
- [Analytics](metrics.md) - Tracks user behavior and system performance

For detailed architecture, see [architecture.md](architecture.md).
