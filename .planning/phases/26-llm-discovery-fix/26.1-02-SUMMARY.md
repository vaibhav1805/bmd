## Decisions Made

- **No confidence capping**: Claude's API already outputs calibrated confidence 0.5-0.95; an arbitrary 0.65 cap was preventing high-confidence discoveries from passing. Removed entirely.
- **SignalLLM threshold 0.50**: LLM signal is qualitatively different — it's reasoning about document purpose and architecture, not counting syntactic patterns. 0.50 is the minimum meaningful Claude confidence.
- **Dual threshold (0.65) > solo threshold (0.60)**: Counter-intuitive but intentional — when two algorithms agree, we can demand slightly higher confidence because the corroboration itself provides signal. Solo gets a lower bar to avoid requiring multi-algorithm agreement for LLM-only discoveries.
