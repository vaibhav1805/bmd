    SetOnChange(fn func(WatchEvent)) — allows Plan 18-03 to inject a notification callback.

    handleEvent(evt WatchEvent) — core logic (unexported):

    For WatchDeleted:
      1. relPath = evt.RelPath
      2. u.index.bm25.RemoveDocumentsByRelPath(relPath)
         (use the exported wrapper: call u.index.UpdateDocuments(nil, []string{relPath}))
      3. Remove graph node: u.graph.RemoveNode(relPath) — add this method to graph.go if absent
      4. Persist graph: if u.db != nil { u.db.SaveGraph(u.graph) }
      5. Re-run registry from updated graph + scan: rebuildRegistry(u)
      6. u.mu.Lock(); u.stats.FilesRemoved++; u.mu.Unlock()
      7. Fire onChange callback if set

    For WatchCreated | WatchModified:
      1. doc, err := LoadDocument(evt.Path, u.dir) — use ScanDirectory approach for single file
      2. If err (file unreadable): u.stats.Errors++; return
      3. If existing doc hash unchanged (check u.index.docMeta[doc.ID].Hash == doc.ContentHash):
         u.stats.FilesSkipped++; return
      4. u.index.UpdateDocuments([]Document{doc}, nil)
      5. Re-extract graph relationships for this file using the existing Extractor:
         ex := NewExtractor(u.graph); ex.ExtractFromDocument(doc)
         Persist graph if db != nil
      6. Rebuild registry (rebuildRegistry)
      7. u.stats.FilesIndexed++
      8. Fire onChange

    rebuildRegistry(u *IncrementalUpdater) (unexported):
      - Scan all docs quickly: docs, _ := ScanDirectory(u.dir)
      - u.registry = NewComponentRegistry()
      - u.registry.InitFromGraph(u.graph, docs)
      - Save registry JSON: SaveRegistry(u.registry, filepath.Join(u.dir, RegistryFileName))

    LoadDocument helper (add to helpers.go if not present):
      Given a single file path and root dir, return a Document with ContentHash populated.
      Use os.ReadFile, compute MD5 hash (same as scanner.go pattern), build Document struct.

    Important: The index's docMeta field is unexported. Access via the existing
    Index.UpdateDocuments method which handles hash comparison internally.
    For the skip-check, use Index.IsStale — if calling with just one file's data,
    create a minimal []Document slice and compare. Or expose a HasChanged(doc Document) bool
    helper on Index. Use whichever approach compiles cleanly.

    NOTE: Graph.RemoveNode may not exist. Check graph.go. If absent, add it:
    ```go
    func (g *Graph) RemoveNode(id string) {
        delete(g.Nodes, id)
        // Remove all edges where Source or Target == id
        filtered := g.Edges[:0]
        for _, e := range g.Edges {
            if e.Source != id && e.Target != id {
                filtered = append(filtered, e)
            }
        }
        g.Edges = filtered
    }
    ```
